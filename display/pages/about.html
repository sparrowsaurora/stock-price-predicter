This Project

This project delivers comprehensive stock data analysis by combining historical prices, real-time market data, technical indicators, and relevant news sources.

A Long Short-Term Memory (LSTM) neural network is used to generate short-term forecasts, estimating the probability of price movement over a 5-business-day window. Predictions are visualized through interactive charts and accompanied by detailed, text-based summaries.

The system is built using Python, FastAPI, and the Alpaca Markets API. Predictions are tested via paper trading to simulate live performance.

# This project serves as the foundation for a larger system designed to autonomously trade and manage multi-asset portfolios.
